# 1. What is Python?

Python is a high-level, interpreted, and easy-to-learn programming language. It was created by **Guido van Rossum** and released in **1991**. Python is widely used in Data Science, Artificial Intelligence, Machine Learning, Web Development, Automation, and Cybersecurity.

# 4. What is a Variable?

A variable is a container used to store data in a program.

### Example

```python
name = "Sahil"
age = 20
```

---

# 5. Variable Naming Rules

- Variable names cannot start with a number.
- Variable names cannot contain spaces.
- Variable names can contain letters, numbers, and underscores (_).
- Python keywords cannot be used as variable names.
- Variable names are case-sensitive.

### Correct Examples

```python
student_name = "Sahil"
age = 20
cgpa = 8.5
```

### Incorrect Examples

```python
1name = "Sahil"
student-name = "Sahil"
class = "Python"
```

# 7. print() Function

The `print()` function is used to display output on the screen.

### Example

```python
print("Hello World")
print("My Name is Sahil")
```

---

# 8. input() Function

The `input()` function is used to take input from the user.

### Example

```python
name = input("Enter Your Name: ")
print("Welcome", name)
```

---

# 9. Comments

Comments are used to explain the code. Python ignores comments during execution.

### Example

```python
# This is a comment

print("Hello")
```

---

# 10. Difference Between print() and input()

| print() | input() |
|----------|----------|
| Displays output | Takes input from the user |
| Used for showing information | Used for receiving information |

---

# 11. Difference Between Integer and String

| Integer | String |
|----------|---------|
| Stores numbers | Stores text |
| Example: 10 | Example: "10" |
